inline void update(const SolvQuery &other)¶

update == union Unites query with other query (aka logical or) Result of the other query is added to result of this query Throw UsedDifferentSack exceptin when other has a different SolvSack from this

inline void intersection(const SolvQuery &other)¶

Intersects query with other query (aka logical and) Keep only common packages for both queries in this query Throw UsedDifferentSack exceptin when other has a different SolvSack from this

inline void difference(const SolvQuery &other)¶

Computes difference between query and other query (aka q and not other) Keep only packages in this query that are absent in other query Throw UsedDifferentSack exceptin when other has a different SolvSack from this
